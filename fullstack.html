Full Stack Development

Learn how websites work from front to back.

What is Full Stack Development?

Full stack development means building both the front part (what users see) and the back part (how it works) of a website. A full stack developer knows how to do both jobs.

Main Parts of Full Stack
  • Front-end: HTML, CSS, JavaScript, React
  • Back-end: Node.js, Python, PHP, APIs
  • DevOps: Git, Hosting, Cloud tools

Front-end Example

This is a simple form that lets users enter their name and email.


<form id="userForm">
  <label>Name: </label>
  <input type="text" id="name" required /><br>
  <label>Email: </label>
  <input type="email" id="email" required /><br>
  <button type="submit">Submit</button>
</form>

<p id="output"></p>

<script>
  document.getElementById("userForm").addEventListener("submit", function(event) {
    event.preventDefault();
    const name = document.getElementById("name").value;
    const email = document.getElementById("email").value;
    document.getElementById("output").innerText = "Name: " + name + ", Email: " + email;
  });
</script>

Back-end Example (Node.js)

This code runs on a server. It gives user data when asked.


// Simple server using Express
const express = require('express');
const app = express();

const users = [
  { id: 1, name: 'Alice' },
  { id: 2, name: 'Bob' }
];

app.get('/api/users', (req, res) => {
  res.json(users);
});

app.listen(3000, () => {
  console.log('Server running on port 3000');
});

Database Example (MongoDB)

This code connects to a MongoDB database, adds a user, and shows all users.


// MongoDB example with Node.js
const { MongoClient } = require('mongodb');

async function main() {
  const client = new MongoClient('mongodb://localhost:27017');

  try {
    await client.connect();
    const db = client.db('myDatabase');
    const users = db.collection('users');

    // Add one user
    await users.insertOne({ name: 'John', email: 'john@example.com' });

    // Show all users
    const result = await users.find().toArray();
    console.log(result);
  } finally {
    await client.close();
  }
}

main().catch(console.error);
