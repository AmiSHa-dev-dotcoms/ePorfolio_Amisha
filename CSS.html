Style your webpage with colors, spacing, and creative layouts.

What is CSS?

CSS stands for Cascading Style Sheets. It helps us design how a webpage looksâ€”like colors, fonts, spacing, and more. With CSS, we turn plain HTML into beautiful web pages.

Simple CSS Example

  background-color: #e3f2fd;
  font-family: 'Segoe UI', sans-serif;
  padding: 20px;

CSS Selectors

Selectors tell CSS which elements to style. You can target tags, classes, or IDs.

Common Selectors

p {
  color: darkblue;
}

.highlight {
  background-color: yellow;
}

#title {
  font-size: 24px;
  text-align: center;
}

.container p {
  line-height: 1.6;
}

Understanding the Box Model

Every HTML element is a box. The box model includes padding, borders, and margins that affect spacing.

Example: Box Style

.box {
  width: 200px;
  padding: 20px;
  margin: 15px;
  border: 2px solid #333;
  background-color: #f8f9fa;
}
